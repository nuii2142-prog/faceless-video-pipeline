# Gemini / Antigravity — start here

Read [`AGENTS.md`](AGENTS.md) first. It is the universal entry point for any AI agent in this repo.
Then read `docs/WORKFLOW.md` (the pipeline) and `docs/visual-style.md` (the locked image style).

Your two jobs in this project: (1) write the narration script, (2) write one image prompt per
scene into `output/<slug>/scene_prompts.json`. Everything else is plain Python scripts you run.
